# TP 3 – Les formulaires en Symfony

## Objectif
Réaliser un formulaire produit en Symfony à partir d’un code HTML fourni, en utilisant :
- Twig pour le rendu
- Les Form Types de Symfony
- Bootstrap 5 pour le style



## Travail réalisé
- Création d’une page produit avec Twig
- Transformation du formulaire HTML en **Form Type Symfony**
- Gestion du formulaire dans un contrôleur
- Récupération et validation des données soumises
- Intégration de Bootstrap 
- Protection CSRF activée

## Fonctionnalités
- Affichage d’un produit (casque audio)
- Champ quantité (1 à 10)
- Sélection de la couleur
- Soumission du formulaire
